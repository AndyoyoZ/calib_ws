单目相机与三维激光（velodyne）的外参数标定说明

1. 首先运行 run_sensors.sh 启动相机和激光雷达
2. 运行 bag_xx.sh 进行数据记录
3. 运行 bag2pcdpng 对bag包的数据进行保存，其中相机数据保存为 ****.png格式的图片，激光数据保存为 ****.pcd格式的点云数据
4. 运行 select_pointcloud　,通过shift+click在点云数据中选取棋盘格在点云中的位置，尽量选择四角。点云分割结果保存为 ****_s.pcd
5. 运行 findchessboard ,从第４步得到的点云文件中找到棋盘格角点对应的点云，从第３步得到的图片文件中提取棋盘格角点，并采用PnP求解相机与激光雷达的外参数矩阵。
6. 待完善：对第５步得到的结果进行优化。（思路：对多帧数据进行BA优化）
